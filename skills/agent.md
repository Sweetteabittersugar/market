# Agent — Agent 路由

任务自动匹配最合适的 Agent。

## 手动指定
```
@architect <任务>   — 架构设计
@coder <任务>      — 写代码
@reviewer <任务>   — 代码审查
@debugger <任务>   — 调试
@explorer <任务>   — 代码搜索
@test-writer <任务> — 写测试
@devops <任务>     — 运维部署
@doc-writer <任务> — 文档
@security-reviewer <任务> — 安全审计
```

## 自动路由
系统根据任务关键词自动匹配。详见 `AGENTS.md`。
