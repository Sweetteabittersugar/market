# Skill 列表

| Skill | 用途 | 触发词 |
|-------|------|--------|
| review | 四维度代码审查 | 审查、review |
| debug | 六步结构化调试 | bug、报错、调试 |
| cost | 费用查询 | `/cost` |
| history | Agent 执行记录 | `/history` |
| help | 使用说明 | `/help` |
| workflow | 9 阶段工作流 | 工作流、切阶段 |
| agent | Agent 路由 | @agent名 |

Skill 是预定义的操作流程，告诉 Claude Code 怎么高效完成特定任务。
