## 🗺 工作流 · 12 阶段

> `source scripts/workflow.sh <stage>` 看当前阶段装备

```
 对话式（人裁决）          层级式（主Agent裁决）
 ─────────────────────────────────────────────────────
 -1 灵感 → 0 需求          1 Spec  → 2 技术 → 3 API → 3.5 故事
   ↓        ↓               ↓         ↓        ↓        ↓
 explorer  pm(协调)     architect  architect  api     pm
                                              designer
 ─────────────────────────────────────────────────────
                           4 后端 → 5 前端 → 6 验证 → 7 视觉 → 8 部署🔴 → 9 复盘
                             ↓        ↓        ↓        ↓        ↓         ↓
                           coder    coder     qa     designer   qa    reviewer
```

| 阶段 | 一句话 | Skill | Agent |
|------|--------|:---:|:-----:|
| -1 | 竞品→问题陈述 | /spec | explorer |
| 0 | 需求→反派挑战→阶段清单 | /spec | pm |
| 1 | 宪章·功能清单·AC | /spec | architect |
| 2 | 三问·可行性·致命问题 | /research | architect |
| 3 | Contract-First·端点·错误格式 | — | api-designer |
| 3.5 | Epic→Story·sprint 更新 | /create-story | pm |
| 4 | TDD→curl→自审 | — | coder |
| 5 | 组件树→三态→截图 | — | coder |
| 6 | 关键路径·契约·门控 | /gate-check | qa |
| 7 | 机械检查→截图→人判 | — | designer |
| 8 | 安全扫描·回归·冒烟 🔴 | /gate-check | qa |
| 9 | 三问→教训→memory | /retro | reviewer |

## 🛒 逛街路线

> 不知道走哪条？对号入座。

| 你要做什么 | 🚶 路线 |
|-----------|---------|
| **新项目启动** | `/spec` → `source workflow.sh -1` → 跟着 12 阶段 |
| **技术选型** | `/research <topic>` → 看报告 → `@architect` |
| **开始写代码** | `/create-story` → Story 文件 → `source workflow.sh 4` |
| **阶段做完了** | `/gate-check` → PASS → commit → `/compact` |
| **大任务结束** | `/retro` → 看复盘 → 教训自动写入 |
| **修 Bug** | `source workflow.sh quick` → `@coder` → `@qa` |
| **不知道用啥** | `source scripts/workflow.sh` 全菜单 |
| **查惯例** | [conventions.md](conventions.md) |
| **查阶段** | [工作流文档](全栈开发工作流.md) |
