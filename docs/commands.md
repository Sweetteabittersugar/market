# 命令参考

## `/cost` — 费用查询

```
/cost               → 今日/本周/本月汇总
/cost --days 7      → 最近 7 天
/cost --month       → 本月明细（按模型）
```

费用数据自动记录，无需手动操作。

## `/history` — Agent 执行记录

```
/history               → 最近 10 条
/history --last 20     → 最近 20 条
/history --agent coder → 按 Agent 筛选
/history --today       → 只看今日
```

记录每次 Agent 派发的任务、文件、状态、费用。

## `/help` — 使用说明

```
/help                  → 列出所有话题
/help commands         → 命令参考（本页）
/help agents           → Agent 列表和用途
/help skills           → Skill 列表和用途
/help workflow         → 工作流详解
/help cost             → 费用追踪说明
```
